# ziko-lucide
Lucide Icons for ZikoJs 

## Installation 

```bash
npm i ziko-lucide 
```

## Usage

```js
import {Activity} from 'ziko-lucide'

Activity({
    stroke : 'darkblue'
})
```

## Props

|Name|Type|Default|
|-|-|-|
|`fill`|`string`|`"none"`|
|`width`|`number`|`24`|
|`height`|`number`|`24`|
|`viewBox`|`string`|`"0 0 24 24"`|
|`stroke`|`string`|`black`|
|`stroke-width`|`number`|`2`|
|`stroke-linecap`|`string`|`"round"`|
|`stroke-linejoin`|`string`|`"round"`|
